Accessibility Statement

Clear Website Structure

The website structure is clear and uses appropriate heading tags (such as h1, h2, etc.). For example, the main title of the home page uses the <h1> tag, and the subheadings use <h2>, <h3>, etc. in sequence to ensure that the heading hierarchy is clear and logical.

This structure not only helps visual users quickly browse content but also enables users who use screen readers to navigate page content by headings.

Clear Labels for Navigation

We ensure that all links and buttons have descriptive labels that clearly indicate their purpose. For example, a button for "Learn more about me" uses "Learn more about me" as the button label instead of the vague description like "Click here."

Alternative Text for Images

All images have appropriate alt text to describe their content. For example, a photo of me take part in a swimming competition uses alt="A photo of me in a swimming competition." This ensures that users who use screen reader can access image information effectively.

Color Contrast and Readability

We have chosen a high-contrast color scheme (Black and White) to ensure a clear distinction between text and background. This allows even users with impaired vision to easily read the content.

Language Definition

We have used the lang="en" language attribute in the HTML file to allow screen readers to accurately identify the language type. This helps with correct pronunciation and reading of the content, enhancing the user's auditory experience.
